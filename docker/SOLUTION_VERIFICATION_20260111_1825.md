## 最终解决方案

在 `Dockerfile` 中添加：

```dockerfile
# 1. 设置缓存目录，避免挂载覆盖
ENV UV_CACHE_DIR=/root/.cache/uv

# 2. 预安装/缓存 MCP 工具
RUN uvx mcp-server-time --help || echo "Pre-cached"
```

## 建议后续优化

1. **生产环境应用**: 确保所有使用的 MCP 工具（如 `mcp-trends-hub`）都已预装。
2. **并发控制**: 虽然不再阻塞，但高并发仍会导致响应变慢，建议在 `rcoder` 层面限制最大并发容器数（如 20-30）。
