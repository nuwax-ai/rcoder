使用说明：
RCoder 代理模式（推荐）：
1. 配置 RCoder 服务地址（默认：http://127.0.0.1:8088）
2. 输入 User ID（例如：user_123）
3. 输入 Project ID（从聊天响应中获取，例如：31e1acc6fbac43ec900da01d0296aa24）
4. 点击"连接 VNC"按钮，将通过 /computer/vnc/{user_id}/{project_id}/vnc.html 访问

直接端口模式：
1. 在容器内，noVNC 服务运行在 6080 端口
2. 查找容器端口映射：docker port <container_name>
3. 输入映射到主机的端口号（例如：如果容器 6080 映射到主机 50001，则输入 50001）
4. 点击"连接 VNC"按钮开始测试
